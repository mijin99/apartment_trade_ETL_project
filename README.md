공공 api를 가져와 호출합니다.
호출시 오류 상황을 피하기 위해 3초 간격으로 총 3번 호출 시도합니다.
전월단위로 호출된 데이터는 json형태로 s3 raw데이터로 저장합니다. 항상 최신데이터 유지
그 후 spark 컬럼추가, 중복제거, 파티션, parquet 형식변환 등의 작업을 마친 후 s3 curated에 재 적재합니다. 재적재시 월 데이터는 항상 최신본을 유지합니다.
해당 데이터는 glue catalog와 athena 조합으로 데이터 조회가 가능하고, grafana로 데이터 지표를 시각화 할 수 있습니다.
현재는 airflow에서 일단위로 호출하게 되어있으며, 실제 적재되는 데이터는 전월데이터로 변경가능성이 있으므로 재적재합니다. 
airflow dag을 통해 수집, 적재, 변환 등의 작업을 자동화 합니다.
해당 실행 로그는 postgresql metadata로 기록됩니다.
